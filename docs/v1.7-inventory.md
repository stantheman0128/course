## Classification key

| Code | Meaning | In Legacy | In Modern |
|---|---|---|---|
| **M** | Must preserve **exactly** (structure, behavior, semantics) | identical | may redesign |
| **D** | Data-update allowed; structure & behavior unchanged | 75 → 73, 114-1 → 114-2, etc. | may redesign |
| **X** | Modern-only change allowed; Legacy must keep v1.7 form | unchanged | may redesign |
| **N** | New addition not in v1.7 (only allowed if explicitly approved) | NOT allowed in Legacy | may exist in Modern |

If a row is unclassified, default is **M**.

## §O. Migration constraints (summary)

The new Legacy theme must:

1. **Be structurally identical to v1.7**: header → sticky stats → content-wrapper (simulator + tree) → footer. No Layout shell, no nav, no Routes wrapper visible to the user.
2. **Default `simulatedCourses` to empty Set**. The user opts in.
3. **Preserve the 4 stats semantics**: 已修 / 畢業 / 還需 / 完成度. No "本學期 pending" card.
4. **Render `earned / required` without clipping**. Overflow must be visible per v1.7.
5. **Include all 4 interactive subsystems**: stellar canvas (§B), scroll-collapse stats (§D6-D10), bottom sheet (§G), fullscreen mode (§K), keyboard shortcuts (§L).
6. **Match v1.7's data model semantics**: `isNew` flag on simulated courses, section-header text swaps based on `114-X` note string, completed-courses collapsed by default with green summary.
7. **Have NO localStorage persistence** in Legacy. Refresh = reset.
8. **NOT claim 115-1 recommendation**: either remove `GapAdvisor`'s wording or downgrade to "未滿足分類的可選課程" (which is what v1.7 does in 「💡 可選課程」 inside the tree, not as a separate panel).

The Modern theme is free to redesign all of this, EXCEPT:
- The Modern theme is also bound by §N9 (no 115-1 claim without real data)
- The Modern theme must not pretend overflow doesn't exist (§N5 applies — display must be honest, even if styled differently)

Data updates allowed in both:
- 110 → 110 cohort, post-112.09.22 amendment (rules unchanged in numbers since amendment matched 111)
- 75 → actual transcript-derived earned
- 114-1 → 114-2 sidebar
- Footer version string (`v2.0` or similar) and "Made with" line may update

Anything else not listed here is **M** (must preserve) by default.
